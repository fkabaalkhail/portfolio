# Portfolio - SEG3125 Assignment 1

Personal portfolio website for SEG3125 (Analysis and Design of User Interfaces), Summer 2026.

## About

This is my portfolio entry page for the course. It has three main sections:
- **About me** - who I am, my background, skills, and interests
- **How I work** - my development process and what I'm learning in SEG3125
- **Case studies** - 4 placeholder links to future design projects (coming soon)

## Tech Stack

- Next.js (React framework)
- Tailwind CSS for styling
- Deployed on Vercel

## Running Locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Deployment

Hosted on Vercel. Auto-deploys on push to master.
